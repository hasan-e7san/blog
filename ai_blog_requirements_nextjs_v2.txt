
AI BLOG PLATFORM - FULL REQUIREMENTS

====================
PROJECT OVERVIEW
====================
Name: AI Blog Platform
Domain: blog.hasan-ehsan.cloud

Purpose:
- Build the project in Next.js
- Blog platform with AI-generated articles
- Daily generation: 1 article for each category
- Dashboard for full control

====================
CORE AI CONTENT RULE
====================

The system must generate AI articles for every active category with AI enabled.

Required rule:
- each active category with AI enabled must receive 1 generated article every day

This requirement applies to:
- technology categories
- story categories
- any future categories added by admin

====================
AI GENERATION REQUIREMENTS
====================

Daily Flow:
- Runs once per day
- For each active category with AI enabled:
  - generate 1 article
  - generate title
  - generate excerpt
  - generate content
  - generate SEO title
  - generate SEO description
  - generate tags
  - generate image prompt
  - generate image
  - save article
  - publish directly or save as draft
  - create log record

Required rule:
- the AI must generate 1 article for each active category with AI enabled every day
- the scheduler must loop through all active AI-enabled categories
- no category should be skipped unless disabled or inactive

Rules:
- 1 article per category per day
- prevent duplicates
- log success/failure
- support retry
- support manual generation from dashboard

====================
SEEDERS
====================

Admin Seeder:
- creates default admin
- password must be hashed
- must not create duplicates when re-run

Category Seeder:
- seeds default categories
- must not create duplicates when re-run
- supports AI enabled flag
- must include technology categories
- must include story categories

Required seeded category groups:

Technology categories:
- AI
- Technology
- Programming
- Web Development
- Mobile Development
- Cybersecurity
- Cloud Computing
- Data Science
- Startups
- Gadgets

Story categories:
- Funny Stories
- Scary Stories
- Mystery Stories
- Adventure Stories
- Romantic Stories
- Drama Stories
- Inspirational Stories
- Kids Stories
- Fantasy Stories
- Short Stories

Seeder behavior:
- all seeded categories should be active by default
- all seeded categories should have AI enabled by default
- slugs must be created correctly
- re-running the seeder must not create duplicates

Seeder Requirements:
- run all seeders together
- run admin seeder only
- run category seeder only
